# CryptoDashboard made with React

Hi! This is a dashboard (under development) where you can see the most famous cryptocurrencies out there, and even add them to your favourite list.

## Installation

First, you have to clone the github repository

**`git clone https://github.com/tobicorradi/crypto-dashboard.git`**

Then, install the dependencies (**you must have NodeJS installed**)

**`npm install`**

Finally, start a local development server to run the app

**`npm start`**


*That's it, enjoy :)*
